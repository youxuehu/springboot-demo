k8s 部署服务步骤：

kubectl apply -f nginx.yaml
kubectl get deployments nginx
kubectl expose deployment nginx --type=NodePort --name=nginx-service
kubectl describe services nginx-service

NodePort:               <unset> 31496/TCP

http://localhot:31496

kubectl delete services example-service
kubectl delete deployment hello-world

强制删除pod
kubectl delete pod mysql-rc-cb86n -n default --grace-period=0 --force

查看namespace
kubectl get namespace

查看job
kubectl get job
查看pod
kubectl get pod
kubectl get pod --show-all
查看pod日志
kubectl logs pod名称

提交任务
kubectl apply -f job.yaml
查看job
kubectl get jobs
查看cronjob
kubectl get cronjobs
删除cronjob
kubectl delete cronjob cronjob-demo

删除replication controller后对应的pod也会随之删除
kubectl delete -n default replicationcontroller mysql-rc


从本地向k8s pod 复制文件
kubectl cp ./dockercompose/mysql/data/sql/nacos-mysql.sql  mysql-rc-wr9c5:/root/nacos-mysql.sql
